Recommended method
1. add two-factor authentication to your google account using,
	https://myaccount.google.com => security => signing in to Google => 2-step verification
2. come back to security page under signing in to Google. now click app password => select app - custom name (give any name) => copy the generated password.
Note: It is one time viewable password. So keep it secure.

for local machine testing:
1. https://myaccount.google.com/lesssecureapps
goto the above link and enable.
2. if ip keeps changing then use following link, https://accounts.google.com/DisplayUnlockCaptcha
